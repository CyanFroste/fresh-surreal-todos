# Fresh + SurrealDB + Deno Todo App

## Prerequisites

- Install [Deno](https://deno.land/manual@v1.26.1/getting_started/installation)
- Install [SurrealDB](https://surrealdb.com/install)

## Usage

- Start SurrealDB

```
deno task start-db
```

- Run app on a local dev server

```
deno task start
```
